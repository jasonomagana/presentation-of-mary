# Presentation of Mary Parish Website

A responsive website built for Presentation of Mary Parish in South Central Los Angeles. This project was developed to provide parishioners with easy access to schedules, information, and community updates.

## Overview

This website was created as a real-world project to serve a local parish community. It focuses on clarity, accessibility, and mobile responsiveness while showcasing modern front-end development practices.

The site includes dynamic schedule rendering, responsive design, and interactive UI features.

## Features

- Dynamic schedule rendering using JSON and JavaScript
- Fully responsive layout (mobile-first considerations)
- Mobile navigation menu with toggle + auto-close behavior
- Scroll-based navbar (hides on scroll down, appears on scroll up)
- Clean UI with modern layout and consistent styling
- Bilingual content (English & Spanish)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- Vanilla JavaScript (DOM manipulation, XHR, async concepts)

## What I Learned

- How to structure a real-world front-end project from scratch
- Working with asynchronous JavaScript (XMLHttpRequest)
- Rendering dynamic data from a JSON file
- Managing UI state (mobile menu, navbar behavior)
- Building responsive layouts using Flexbox and Grid
- Improving UX through interaction patterns
